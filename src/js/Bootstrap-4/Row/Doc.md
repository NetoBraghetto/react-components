# Row

## Usage

```jsx
<Row>
    <Col xs="12">Content</Col>
</Row>
```

## Props

|Prop|Description|Values|Default|
|---|---|---|---|
|className|Extra classes|string|''|

## Examples

```jsx
import { Row, Col } from 'rtdigital-react-components';

<Row>
    <Col sm="4">Col 1</Col>
    <Col sm="4">Col 2</Col>
    <Col sm="4">Col 3</Col>
</Row>

```
